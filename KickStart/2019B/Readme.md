## C

- 想歪了，没想到前缀和这个东西
- 我们枚举左端点，那么他右边每个数的贡献是$\lbrace -1, 0, 1 \rbrace$之一
- 我们要求的是左端点开始的最大前缀和
- 从右向左枚举左端点，线段树维护前缀和即可
- 因为每次修改
    - 最多把一个数的贡献从$0$改成$1$
    - 最多把一个数的贡献从$1$改成负值
    - 最多把一个数的贡献从负值改成$0$
- 所以复杂度一共是$O(n \log n)$
