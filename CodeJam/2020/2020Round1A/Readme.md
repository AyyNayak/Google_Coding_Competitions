## A

- 我们先考虑每个串的前后缀(星号前，星号后)
- 这些部分一定要严格匹配，作为约束
- 在考虑中间部分，显然每个串的中间部分要么是一个星号，要么是两端是星号的子串
- 这意味着中间部分只要包含非星号部分作为子串就一定能匹配
- 把所有串中间的非星号拼起来作为中间部分
- 最后把前缀+中间+后缀合并输出即可
